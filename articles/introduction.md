## 💻 ソロ開発エコシステム (The Solo Stack)

**「資金ゼロ・友達ゼロ」**なので、AIエージェントを味方につけて**一人で組織**をやってます。

- **Research**: Perplexity で arXiv論文検索 (週20本実装)
  - **Obsidian Ecosystem**:
    - **RAG**: 蓄積した知識をローカルLLMで検索・対話。
    - **Integration**: **Zotero** (論文管理) + **Longform** (執筆) + **Pandoc** (出力) で研究サイクルを完全自動化。
- **Coding**: **Claude Code** + **Cursor** + **Antigravity**
- **Design**: Figma

### 🔧 Deep Tech (ガチでやってること)

ポンコツなのは人間性だけで、コードは少しだけ本気です。

- **Kernel Construction**: **Rust** からFFIで **CUDA** (学習時) / **Metal** (推論時) を直接制御し、ゼロからカーネル構築。
- **AI Architecture**:
  - **Native Multimodal**: 視覚・聴覚を統合したマルチモーダルAIをフルスクラッチ開発。
  - **Training**: **Lambda Labs** / **RunPod** のクラウドGPUで学習。
  - **Data**: **Hugging Face** でライセンス（Apache 2.0 / MIT等）を厳密に精査してデータセット構築。
- **Backend / Security**: **Rust (Cargo workspace)** で構築。暗号化バイナリでセキュリティを担保。
- **DevOps / Infra**:
  - **Automation**: **n8n** でワークフローを自動化。
  - **Self-Hosting**: **Hetzner** (CPX11) + **Cloudflare** + **Coolify** で格安かつ堅牢なインフラを構築経験あり。
- **Client**: SPM / Gradle を駆使したネイティブUI実装。
